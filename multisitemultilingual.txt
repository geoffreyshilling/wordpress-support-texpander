ultisite can be a helpful way to set up multilingual sites. Multisites is more complicated than single site WP however, so you will want to look at the skills required to create and manage a multisite network before diving in. There are <a href="https://wordpress.org/support/article/create-a-network/">step-by-step instructions</a> for setting up multisite, but you might also find <a href="http://wplang.org/wordpress-multisite-multilingual/">this article about setting up a multilingual network</a> helpful.
